# Founder OS

Founder OS is a founder-focused personal operating system built with React, Vite, Tailwind CSS, Chart.js, and localStorage.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Features

- Dashboard command center with founder score, timer, notes, and follow-ups
- Daily execution tracker with calendar view
- Idea Lab, Opportunity Radar, Networking CRM, Weekly Review, Decision Journal, Knowledge Vault
- KPI analytics with weekly trend charts
- Quick capture via `Q`
- Evening journal modal
- JSON backup export and restore import
- Dark mode support
